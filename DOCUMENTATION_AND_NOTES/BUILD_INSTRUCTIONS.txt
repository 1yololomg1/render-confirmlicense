TROUBLESHOOTING NUITKA BUILD:
=============================

If build fails, check:

1. Missing Dependencies:
   - Ensure all Python packages are installed: pip install -r requirements.txt
   - Check for missing system libraries (Windows SDK, Visual C++)

2. Module Import Errors:
   - Nuitka may need explicit --include-module flags
   - Check build_nuitka.bat for all included modules
   - Add missing modules to the build script

3. Protection Module Issues:
   - Protection module now works with both PyInstaller and Nuitka
   - If issues persist, check protection_module.py for path validation

4. Large Executable Size:
   - Normal for standalone builds (includes Python runtime)
   - Use --onefile for single executable
   - Consider --standalone for folder distribution

5. Runtime Errors:
   - Test the executable before distribution
   - Check logs in ~/.confirm/confirm.log
   - Verify all data files are included if needed
